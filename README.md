# Smart Task Manager

## Overview

Smart Task Manager is a Streamlit-based productivity application that helps users organize and track daily tasks efficiently.

## Features

* Add tasks
* Delete tasks
* Mark tasks as completed
* Priority management
* Category management
* Task analytics dashboard
* Interactive charts
* CSV-based task storage

## Technologies Used

* Python
* Streamlit
* Pandas
* Plotly
